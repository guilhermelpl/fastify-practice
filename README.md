# fastify-practice

# Clonar aplicação 
- git clone https://github.com/guilhermelpl/fastify-practice.git

# Instalação de dependências
- npm install

# Rodar aplicação
- npm run dev

# Comando para criar a tabela no banco de dados:
- node src/scripts/create-table.js
ou
- cd src/scripts
- node create-table.js
